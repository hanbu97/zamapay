# CLI Installer Route Architecture

## Tree

```text
apps/web/app/install.sh
|-- route.ts          # Public CLI installer script endpoint
`-- ARCHITECTURE.md   # Route note
```

## Decisions

- `/install.sh` is the stable CLI installation URL. Today it requires `--from-source`; after releases exist the same URL can install prebuilt binaries.
- The route returns script text only. It does not inspect sessions, credentials, cookies, or merchant project state.
- Installer script bodies are generated by `apps/web/app/install-scripts.ts` so CLI and skill installer routes keep one script-building boundary.
