# ZamaPay Skill Installer Route Architecture

## Tree

```text
apps/web/app/.well-known/skills/zamapay/install.sh
|-- route.ts          # Public shell installer for the ZamaPay Codex skill
`-- ARCHITECTURE.md   # Route note
```

## Decisions

- The installer fetches `/.well-known/skills/zamapay` and writes `SKILL.md` into a local skill directory.
- It does not embed secrets, project ids, session cookies, or webhook material.
- The script body is generated by `install-scripts.ts` so the public page, tests, and route share one implementation.
